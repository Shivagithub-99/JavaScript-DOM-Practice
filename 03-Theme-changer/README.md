# 🎨 Theme Switcher

A simple JavaScript DOM project that allows users to switch between Light, Dark, and Sunset themes using buttons.

## 📌 Features

- Switch between three themes
- Update the current theme dynamically
- Change the card appearance using CSS classes
- Uses a single event listener for all buttons

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5 (Card Component)

## 📚 Concepts Practiced

- DOM Selection (`querySelector`)
- Event Handling (`addEventListener`)
- Event Object (`e.target`)
- `classList.add()`
- `classList.remove()`
- Loops (`for...of`)
- Conditional Statements (`if...else`)
- Dynamic Text Updates (`textContent` / `innerText`)

1. Clone the repository.
2. Open the `03-theme-switcher` folder.
3. Open `index.html` in your browser.
