# ⏰ Digital Clock

A simple JavaScript DOM project that displays the current time and date in real-time. The clock updates every second and supports a 12-hour format with AM/PM, along with a greeting based on the time of day.

## 📌 Features

- Live digital clock
- 12-hour time format (AM/PM)
- Displays current day, month, and year
- Greeting based on the current time
- Leading zeros for single-digit values
- Responsive and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Practiced

- `new Date()`
- `setInterval()`
- `getHours()`
- `getMinutes()`
- `getSeconds()`
- `getDay()`
- `getMonth()`
- `getFullYear()`
- Arrays
- `padStart()`
- DOM Manipulation (`textContent`)
- Conditional Statements (`if...else`)
- Template Literals
