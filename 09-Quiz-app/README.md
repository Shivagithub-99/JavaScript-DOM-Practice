# 📝 Quiz App

A simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.  
Users can answer multiple-choice questions, get their score calculated, and test their knowledge of web development concepts.

## 🚀 Features

- Multiple-choice questions
- One option selection using radio buttons
- Dynamic question loading
- Answer validation
- Score calculation
- Prevents moving forward without selecting an option
- Simple and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 📂 Project Structure

```
Quiz-App/
│
├── index.html
├── style.css
└── script.js
```

## ⚙️ How It Works

1. Questions are stored as JavaScript objects in an array.
2. The first question is displayed when the application loads.
3. Users select one option from the available choices.
4. On clicking the **Next** button:
   - The selected answer is checked.
   - The score is updated if the answer is correct.
   - The next question is displayed.
5. The final score is displayed after completing the quiz.

## 🧠 Concepts Practiced

- DOM Manipulation
- querySelector and querySelectorAll
- Event Listeners
- Arrays of Objects
- forEach() method
- Radio Button Handling
- Conditional Statements
- State Management
