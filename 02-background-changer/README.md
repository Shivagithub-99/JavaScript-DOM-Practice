# 🎨 Background Color Changer

A simple JavaScript DOM project that changes the background color of a box when different color buttons are clicked.

## 📌 Features

- Change the box background color
- Display the selected color name
- Multiple color options
- Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Selection (`querySelector`)
- Functions
- Function Parameters
- Button Click Events (`onclick`)
- Dynamic Styling (`style.backgroundColor`)
- Updating Text using `textContent`

## 📂 Folder Structure

```
02-background-color-changer/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 How to Run

1. Clone the repository.
2. Open the `02-background-color-changer` folder.
3. Open `index.html` in your browser.
