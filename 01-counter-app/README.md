# Counter App

A simple counter application built using HTML, CSS, and JavaScript.

## Features

- Increment counter
- Decrement counter
- Reset counter
- Prevents negative values
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Concepts Practiced

- DOM Selection (`getElementById`)
- Updating DOM using `textContent`
- Functions
- Variables
- Conditional Statements
- Inline `onclick` Events

```

└── README.md
```

- Replace `onclick` with `addEventListener()`
