# 📝 To-Do List

A simple JavaScript DOM project that allows users to add, complete, and remove daily tasks. This project focuses on creating and manipulating DOM elements dynamically using JavaScript.

## 📌 Features

- Add new tasks
- Remove tasks
- Mark tasks as completed (double-click)
- Prevent empty tasks from being added
- Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Practiced

- DOM Selection (`querySelector`)
- Event Handling (`addEventListener`)
- Reading Input Values (`value`)
- Creating Elements (`createElement`)
- Adding Elements (`appendChild`)
- Removing Elements (`remove`)
- CSS Class Manipulation (`classList.add`, `classList.toggle`)
- Dynamic DOM Manipulation

1. Clone the repository.
2. Open the `04-todo-list` folder.
3. Open `index.html` in your browser.
